one time commands for running project

command to install all requirements:

>>pip install -r requirements.txt

installs all packages specified in requirements 

to install a specific package:
>>pip install packagename

incase of any errors while running above commands add this to continue:
example:
>>python -m pip install -r requirements.txt
 or to Run any command
>> python -m command



to run the application:

>>streamilt run mtp.py

mtp .py is the file name
to run anystreamlit file :
>>streamlit run file.py

incase of errors like streamlit doesn't exists:
>> python -m streamlit run file.py
